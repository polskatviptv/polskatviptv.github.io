Tutaj wyjaśniamy, w jaki sposób możesz korzystać z dostępu do aplikacji Set IPTV.


Wprowadzenie Set IPTV
Z aplikacji Set IPTV można korzystać w telewizorach SAMSUNG Smart TV z systemem Tizen OS, LG Smart TV z WebOS, a także na urządzeniach z systemem Android. Z Set IPTV otrzymasz bezpłatną 7-dniową wersję próbną po pierwszej instalacji. Aby w pełni aktywować aplikację Set IPTV, musisz dokonać jednorazowej płatności w wysokości 14,99 €. Aby aktywować swój adres MAC, kliknij TUTAJ .

Jak skonfigurować SET IP
Użyj M3U

Po zainstalowaniu aplikacji Set IPTV na urządzeniu, musisz najpierw znaleźć swój adres MAC. Kliknij TUTAJ, aby przesłać swoją listę.


Krok 1: Wprowadź adres MAC swojego urządzenia w polu „ TV MAC ”.

Krok 2: Teraz musisz wpisać swój M3U w polu „ M3U URL ”

Krok 3: Po wypełnieniu pól „TV MAC” i „M3U URL” wystarczy rozwiązać captcha.

Krok 4: Naciśnij „Wyślij”, aby wysłać listę M3U do urządzenia.
